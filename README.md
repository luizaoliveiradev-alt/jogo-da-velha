# TouchTacToe

Jogo da Velha híbrido que combina peças físicas e interface digital.

## Objetivo

Permitir que jogadores utilizem peças físicas sobre a tela do smartphone para realizar jogadas em um tabuleiro digital.

## Como funciona

A peça X possui quatro pontos touch.

A peça O possui três pontos touch.

Ao posicionar uma peça sobre uma casa do tabuleiro, o sistema identifica o padrão de toque e desenha automaticamente o símbolo correspondente.

## Tecnologias

- HTML5
- CSS3
- JavaScript ES6
- Touch Events API

## Arquitetura

O sistema é dividido em:

- Interface do usuário
- Detecção de toque
- Lógica do jogo
- Verificação de vitória

## Funcionalidades

- Tabuleiro 3x3
- Reconhecimento de peças físicas
- Alternância automática de jogadores
- Detecção de vitória
- Reinício da partida

## Roadmap
